# 首页滚动收缩与训练摘要设计

## 目标

降低顶部时间线的视觉权重；在训练列表滚动时，将器械区收缩为单行磨砂浮层，为今日训练腾出空间；合并展示相同的训练组数据。

## 交互

- 时间线始终固定在顶部，使用更细、更浅的文字。今天采用 Dock 激活态同款浅蓝背景和蓝色文字。
- 监听今日训练列表的滚动位置：大于 8px 时收缩器械区；仅当回到 0 时重新展开。
- 展开态保留当前 2×3 九宫格与分页；收缩态仅保留当前页前三个器械、隐藏分页并使用单行布局。
- 收缩器械区位于列表之上，采用半透明磨砂背景，列表内容可在其下方滚动。

## 数据展示

- 为同一动作的记录按次数、重量、时长与距离分组。
- 每组相同数据合并为 `N组*数据`，例如 `2组*12次*12kg`。
- 不同数据组合以 ` / ` 连接；动作点击和详情编辑逻辑保持不变。

## 验证

- 为组别摘要增加单元测试，并为滚动折叠状态增加静态页面测试。
- 运行全量测试与生产构建。
