# 首页紧凑布局设计

## 目标

以 iPhone SE 的窄高屏幕为下限，压缩首页顶部时间线、器械九宫格和底部 Dock，让用户更快进入今日训练列表。

## 布局规则

- 顶部仍展示 7 天；非当天日期去除卡片背景，仅保留紧凑文字，当天以小胶囊突出。
- 时间线整体高度约 56px，天数不因压缩而减少。
- 器械区域保留 2×3 九宫格与翻页能力；轮播高度降至约 168px，并同步缩小卡片间距、图标和内边距。
- Dock 高度降至约 54px，缩小图标、标签和页面底部预留空间。
- 在最大高度 667px 的媒体查询内进一步减少首页区块上下间距，确保今日训练标题与至少一条记录可见。

## 约束与验证

- 不改变器械点击、分页、今日训练编辑和数据逻辑。
- 桌面与较高屏幕保持舒适间距；紧凑规则只影响小高度屏幕。
- 增加静态布局测试，运行全量测试和生产构建。
